Debian sid 下使用Knetworkmanager

想用鼠标加入无线网。。。仅此而已。。。
sudo apt-get install network-manager-kde

保证在/etc/network/interfaces里只有
auto lo
iface lo inet loopback
别的网卡都要注释掉
确保你在netdev组内，如果不是，则:
sudo adduser your_username netdev



重新起动network-manager服务



sudo /etc/init.d/network-manager restart







运行knetworkmanager应该OK了
