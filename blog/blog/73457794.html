slackware 12 上 安装 oracle 10g express

     为了google这个，乌鸦居然到了thai文和葡萄牙文的blog鸟－ －
主要还是参考那个thai文的－ －http://nattapol.codingmania.com/serendipity/index.php?/archives/13-aaaaaaa-Oracle-Database-10g-Express-Edition-aa-Slackware-12.html
下面是root下执行的指令列
建立用户
groupadd dba
useradd -g dba -s /bin/bash -d /usr/lib/oracle/xe -m oracle
passwd oracle

内核参数我懒得改了，跳了，反正自己一个人用。


rpmfind.net上搜索libaio-0.3.96-3.src.rpm
rpm2tgz libaio-0.3.96-3.src.rpm
反正解开rpm
进去
make
make install

然后安装oracle
rpm -ivh oracle-xe-univ-10.2.0.1-1.0.i386.rpm --nodeps

su - oracle
vim ~/.oraenv 加入
 
#!/bin/sh
ORACLE_HOME=/usr/lib/oracle/xe/app/oracle/product/10.2.0/server
LD_LIBRARY_PATH=${ORACLE_HOME}/lib
ORACLE_SID=XE
PATH=${ORACLE_HOME}/bin:$PATH
export ORACLE_HOME ORACLE_SID PATH
vim ~/.bash_profile加入
#!/bin/bash
. ~/.oraenv
回到root
exit
chown -R oracle:dba /usr/lib/oracle/xe
vim /etc/init.d/oracle-xe
输入命令
:%s/-s \/bin\/bash//g
:%s/> \/dev\/null/# > \/dev\/null/g
:wq

最后一步
/etc/init.d/oracle-xe configure
如果提示以前已经配置，删除/etc/default/oracle-xe就好了


没有rpm,没有deb,麻烦的事情，挺多。
