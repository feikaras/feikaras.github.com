Inotia3 艾诺迪亚3 修改物品的方法。彩虹水晶，技能点重置，16格包包，入场券。

物品ID大全翻墙看这里：http://inotia3.free.fr/items.html

998以下的物品ID*64就是游戏修改需要的数值(后详述)，大于998貌似改不了

思路，得出第一个物品栏格子的地址，修改物品id

步骤：

1.去杂货店买2瓶红，一瓶小红一瓶中红。把小红放在物品栏第一格。存档，退到主菜单，读档。

2.进游戏后，用gamecih2搜索数值320 （物品id5*64），把中红放在物品栏第一格。存档，退到主菜单，读档。

3.再搜索数值384（物品id65*64），不出意外结论只有一个。

4.查看物品ID，修改这个数值为你想要的物品。16格包的话，推荐修改一个Adventure Set，输入数值62080（ID970*64）。

5.如果需要修改其他物品。找一个替死鬼物品丢第一格，重复上述步骤。。。物品数值会被继承。所以买99个小红的话，修改能得到99个其他物品。。


推荐物品：

ID 7     Elixir     元气恢复药水 （hp mp 加满）

ID 719  Awakening Emblem 觉醒徽章

ID 812 Dice of Chaos  混沌骰子。

ID 830 Skill Reset  技能点重置

ID 932 Rainbow Crystal  彩虹水晶

ID 933-935  各种材料套装。

ID 973 Consecrated Enhancement Set   受祝福的增强套装（武器和防具增强卷轴）

 

其他各种自己研究吧。。。970-997都是必备的一些好东西。BOSS和宝物入场券等。
