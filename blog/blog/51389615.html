安装无冬之夜 for linux

今晚刚安装好无冬之夜for linux,来写一下过程。。

参考：
http://forum.ubuntu.org.cn/viewtopic.php?p=429053
http://www.odyguild.net/bbs/thread-2879-1-1.html
http://nwn.bioware.com/downloads/linuxclient.html

这里以无冬之夜钻石版为例子：

1.去官方网站下载Linux客户端文件(1.12GB).
http://files.bioware.com/neverwinternights/updates/linux/nwresources129.tar.gz
   下载客户端补丁文件http://nwdownloads.bioware.com/neverwinternights/linux/gold/nwclientgold.tar.gz
http://nwdownloads.bioware.com/neverwinternights/linux/161/nwclienthotu.tar.gz
   下载官方英文Linux升级包http://nwn.bioware.com/support/patch.html

 2.
解压缩nwresources129.tar.gz
挂载DVD光盘,解压光盘里的Data_Shared.zip和Data_linux.zip文件。再解压光盘data目录下的XP1.zip和XP2.zip文件
解压缩nwclientgold.tar.gz和nwclienthotu.tar.gz
解压缩补丁。。

3.在终端里运行:
./fixinstall


4.编辑nwn
如果声音卡，如注释所说，去掉./lib的路径就好了
# If you do not wish to use the SDL library included in the package, remove
# ./lib from LD_LIBRARY_PATH
export LD_LIBRARY_PATH=./miles:$LD_LIBRARY_PATH

5.进入游戏,在终端里运行:
./nwn


附加，如果要做启动器，下载图标
http://www.kde-look.org/content/show.php?content=5492
