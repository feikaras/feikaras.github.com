Ubuntu 12.04 ibus-pinyin 以及ibus-mozc日文输入 安装

sudo apt-get install ibus ibus-gtk ibus-gtk3 ibus-qt4 ibus-mozc ibus-pinyin

然后打开ibus的首选项，输入法tab里面，自定义活动输入法。下拉列表的汉语和日语必须点小箭头才能展开，我点了半天字没反应还以为bug了

 

ibus如果出现部分程序输入法光标不跟随往往是没有装对应的immoudle比如ibus-gtk ibus-qt4之类

至于论坛上的export GTK_IM_MODULE之类的环境变量实际上不需要做。因为早就写在了xinput.d的目录里了

env | grep IM_MODULE来看看是不是都是ibus模块，而不是xim。
